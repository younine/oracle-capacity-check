# 프로젝트 컨텍스트

## 목적
OCI 무료 티어 인스턴스(VM.Standard.E2.1.Micro) 자동 생성 자동화

## 파일 구조
- `create_instance.py` : OCI SDK로 인스턴스 생성 시도 (재시도 포함)
- `requirements.txt` : Python 의존성
- `.github/workflows/create-instance.yml` : 30분마다 실행되는 GitHub Actions

## 규칙
- 인증 정보는 절대 하드코딩 금지, 환경변수로만 처리
- 실패 시 exit(1) → Actions가 다음 스케줄에 재실행
- 성공 시 exit(0) → Actions 정상 종료

## OCI 타겟
- Shape: VM.Standard.E2.1.Micro
- Region: ap-seoul-1
- OS: Ubuntu 22.04

## 필요한 GitHub Secrets
| Secret 이름 | 설명 |
|---|---|
| OCI_USER_OCID | OCI 콘솔 → 프로필 → User Settings |
| OCI_TENANCY_OCID | OCI 콘솔 → 프로필 → Tenancy |
| OCI_FINGERPRINT | ~/.oci/config 의 fingerprint 값 |
| OCI_REGION | 예: ap-seoul-1 |
| OCI_PRIVATE_KEY | ~/.oci/oci_api_key.pem 전체 내용 |
| OCI_COMPARTMENT_ID | oci iam compartment list 로 확인 |
| OCI_SUBNET_ID | oci network subnet list 로 확인 |
| OCI_IMAGE_ID | oci compute image list 로 확인 |
| OCI_SSH_PUBLIC_KEY | ~/.ssh/id_rsa.pub 전체 내용 |
| OCI_AVAILABILITY_DOMAIN | oci iam availability-domain list 로 확인 |
